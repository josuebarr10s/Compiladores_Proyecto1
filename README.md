🍽️ Compilador para Sistema de Restaurante
📌 Descripción

Este proyecto consiste en el desarrollo de un compilador especializado para un lenguaje diseñado con el propósito de gestionar operaciones dentro de un restaurante.
El sistema permite interpretar instrucciones relacionadas con pedidos, menú, clientes y facturación mediante un lenguaje estructurado propio.

El compilador realiza las fases clásicas:

Análisis léxico
Análisis sintáctico
Análisis semántico
Generación de resultados / ejecución
🎯 Objetivos
Diseñar un lenguaje específico para la administración de restaurantes
Implementar un compilador capaz de validar y ejecutar instrucciones
Detectar errores léxicos, sintácticos y semánticos
Simular operaciones reales como pedidos, cuentas y gestión de menú
